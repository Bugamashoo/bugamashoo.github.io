Class: DefaultContactListener

.Box2D. DefaultContactListener

new DefaultContactListener()

This class implements a contact listener. The default behaviour is to check if the two bodies that contacted had a callback set up by one of the following: Phaser.Physics.Box2D.Body.createBodyContactCallback Phaser.Physics.Box2D.Body.createCategoryContactCallback Phaser.Physics.Box2D.Body.createFixtureContactCallback
