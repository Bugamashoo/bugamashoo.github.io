Class: PointProxy

.Box2D. PointProxy

PointProxy

new PointProxy(world, object, gettor, settor)

A PointProxy is an internal class that allows for direct getter/setter style property access to box2d.b2Vec2 objects but inverses the values on set. The value of the vector is not stored in this class. Instead, it holds a reference to the object that actually stores the value, along with the functions used to get and set from that object.

Parameters:
Name Type Description
world Phaser.Physics.Box2D

A reference to the Box2D world, used for pixel/meter conversions.
object any

The object to bind to, which holds the actual value.
gettor function

The function of the bound object which gets the value from it.
settor function

The function of the bound object which sets the value in it.
